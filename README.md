# Sikh Trivia (Web)

A simple Sikh trivia quiz built with Next.js and React.

## Tech stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Features

- Multiple-choice Sikh trivia questions
- Answer feedback with correct/incorrect states
- Score tracking
- Mobile-friendly layout

## Running locally

```bash
npm install
npm run dev
# then open http://localhost:3000
